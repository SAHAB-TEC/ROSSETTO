The hierarchy of accounts defines how accounts are related to one another. This module will visually add the parent id of each account and build a tree structure relationship between accounts.

Installation

- Copy odoo_account_hierarchy module to addons folder

- Install the module normally like other modules

Configuration

Enable the Show Full Accounting Features and Show Account Hierarchy options.



Under Accounting, you may now access the Chart of Accounts Hierarchy menu.



Update Parent Account for each Account in the Chart of Accounts option under Accounting settings.



Open Chart of Account Hiearchy Wizard
